* **Exam Tips:**
  * Uses schema on read technology.
  * Totally serverless.
  * Pay for what you read.
  * Doesn't alter source data.
  * Supported data formats:
    * XML
    * JSON
    * CSV
    * TSV
    * AVRO
    * ORC
    * PARQUET
